# canzoniere-san-marco
Canzoniere digitale della Parrocchia San Marco di Udine
## Setlist e login Google
- Senza login: una sola setlist salvata sul dispositivo.
- Con login Google: più setlist personali salvate in Firestore e sincronizzate.
- Al primo login viene proposta l'importazione della setlist locale.
- Pubblicare anche `firestore.rules` prima del test cloud.
