# Documentazione

Questa cartella conserva il lavoro di controllo e le note tecniche del canzoniere.

## Revisione dei canti

- `revisione-canti-001-010.md` fino a `revisione-canti-151-159.md`: controlli svolti a blocchi di dieci canti;
- `revisione-completa-001-159.md`: riepilogo della revisione complessiva;
- `aggiornamento-autori-sottotitoli.md`: interventi su titoli, sottotitoli e autori.

## Ricerca e tag

- `tag-struttura-v1.md`, `tag-ui-v1.md`, `tag-render-fix.md`: struttura e presentazione dei tag;
- `ricerca-tag-v1.md`, `ricerca-tag-integrata-v2.md`: evoluzione della ricerca;
- `tag_tempi_liturgici_v1.md`, `suggerimenti_liturgici_v2.md`: suggerimenti per tempi e celebrazioni;
- `integrazione-tag-canti-per-temi.md`: integrazione dei temi.
