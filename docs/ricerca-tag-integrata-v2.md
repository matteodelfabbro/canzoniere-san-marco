# Ricerca con tag integrati

I tag ora sono pensati come suggerimenti separati dalla ricerca testuale.

Ordine:
1. risultati per titolo/testo
2. suggerimenti tematici dai tag

Esempi:
- matrimonio -> Canti per Matrimonio
- natale -> Canti per Natale
- comunione -> Canti per Comunione

I tag non sostituiscono la ricerca normale.
